# S. T. Kirchberg | Personal Portfolio

A sleek, terminal-inspired personal website built with a focus on a arch linux aesthetic, specifically tailored for a Linux enthusiast. It features a bilingual interface (EN/DE), responsive design across four breakpoints, and an interactive UI.

---

## Features

* **Terminal-Inspired UI**: Clean, mono-spaced design featuring a command-line style header and ASCII art logo.
* **Bilingual Support (i18n)**: Fully integrated English and German translation system using a custom JavaScript implementation and data-i18n attributes.
* **Dynamic Navigation**: Interactive sidebar with a smart scroll-spy indicator that tracks the current section.
* **Responsive Architecture**: Four distinct CSS tiers for seamless viewing: PC, Laptop, Tablet, and Mobile.
* **Interactive Effects**: Features a radial mouse glow following the cursor and stylized project windows.

## Tech Stack

* **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **Design**: Custom CSS variables for a dark, Tokyo Night-inspired color palette.
* **Icons**: SVG-based social icons and Skill Icons for tech stack visualization.

## Structure

* **index.html**: Main structure and language keys.
* **styles/**: Contains breakpoint-specific CSS files (PC, Laptop, Tablet, Mobile).
* **scripts/index.js**: Logic for translations, navigation indicators, and mouse effects.
* **images/**: Favicon and project screenshots.

## Customization

### Adding Translations
To add or modify text, update the translations object in scripts/index.js. Ensure the data-i18n attribute in the HTML matches the key in the JavaScript object.

# S. T. Kirchberg | Personal Portfolio

Eine moderne, Terminal-orientierte persönliche Website, die mit Fokus auf eine Arch-Linux-Ästhetik entwickelt wurde. Sie richtet sich speziell an Linux-Enthusiasten und bietet eine zweisprachige Benutzeroberfläche (EN/DE), ein responsives Design über vier Breakpoints sowie eine interaktive UI.

---

## Features

* **Terminal-Design**: Ein klares, einspaltiges Design mit einem Header im Befehlszeilen-Stil und einem ASCII-Art-Logo.
* **Zweisprachige Unterstützung (i18n)**: Ein vollständig integriertes Englisch- und Deutsch-Übersetzungssystem, das über eine benutzerdefinierte JavaScript-Implementierung und data-i18n-Attribute gesteuert wird.
* **Dynamische Navigation**: Interaktive Sidebar mit einem intelligenten Scroll-Spy-Indikator, der den aktuellen Bereich der Seite verfolgt.
* **Responsive Architektur**: Vier separate CSS-Ebenen für eine optimale Darstellung auf PC, Laptop, Tablet und Mobilgeräten.
* **Interaktive Effekte**: Ein radialer "Mouse-Glow", der dem Cursor folgt, sowie stilisierte Projektfenster im Linux-Stil.

## Tech Stack

* **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **Design**: Verwendung benutzerdefinierter CSS-Variablen für eine dunkle Farbpalette.
* **Icons**: SVG-basierte Social-Media-Icons und Skill-Icons zur Visualisierung des Tech-Stacks.

## Struktur

* **index.html**: Hauptstruktur der Website und Definition der Sprachschlüssel.
* **styles/**: Enthält die Breakpoint-spezifischen CSS-Dateien (PC, Laptop, Tablet, Mobile).
* **scripts/index.js**: Logik für die Übersetzungen, Navigationsindikatoren und Maus-Effekte.
* **images/**: Favicon und Screenshots der Projekte.

## Anpassung

### Übersetzungen hinzufügen
Um Texte zu ändern oder hinzuzufügen, muss das `translations`-Objekt in der Datei `scripts/index.js` aktualisiert werden. Dabei ist darauf zu achten, dass das `data-i18n`-Attribut im HTML mit dem entsprechenden Schlüssel im JavaScript-Objekt übereinstimmt.

## Lizenz
Dieses Projekt ist Open-Source. Es kann gerne als Vorlage für eigene Portfolios im Terminal-Stil verwendet werden. Mir ist es vollkommen egal.
